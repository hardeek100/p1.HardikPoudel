Microwave User Interface

Microwaves oven are the kitchen appliances used to heat food. It has been around us most of our lives. Thousands of companies sell microwaves with different designs but most of the microwaves that we see either have too many buttons or unnecessary buttons. The user interface has to be clear and functional.

Take this microwave from my workplace for an example. The buttons on the user panel are confusing and unnecessary. The door switch is not labeled and has no power switch. Eldery people will have trouble figuring out the buttons on this microwave. If there are childrens in the house they can push random buttons on the microwave and it can be running for hours and can waste energy. Because the door only opens from from right to left, left handed people will have to spend some time getting used to it.

I mostly use microwave to reheat the leftover foods, cook popcorn or rarely to defrost the food and cook pizza. The interface from the example microwave has only popcorn button and quick 30s buttons that are helpful. The other buttons are unnecessary or confusing and I might never press it. The common sequence of user interaction with the microwave are:

  • 1. Turn power on
  • 2. select the food or set timer
  • 3. open door
  • 4. place food
  • 5. close door
  • 6. start
  • 7. wait for food
  • 8. open door
  • 9. take food
  • 10. close door
  • 11. turn power off

    • The microwave provides a user feedback on the display panel. The user control panel has buttons labeled for the desired options and number pad to set the desired time. The food compartment light lets the user know if the microwave is running or not.

    With the current design people might make a common mistake by pressing wrong buttons, search for power button and door switch. ````````````````````````````````````````````````````````````````````````````````````````````````````````

    Improvements

    There are some improvements that can be made for the control interface.

  • Use food icons for the extra buttons
  • Insert power button
  • Insert door switch
  • Open door upwards so that its helpful for both right or left handed users

    • This is a microwave design I came up with. The screen size is 1000*500. It contains all the buttons for options that microwave is mostly used for. Instead of label, I used icons for extra buttons so that it becomes less confusing for the users. The mirowave has power button to save power. It has a door switch that displays the current door status and the door opens upward unlike the traditional microwaves. This will let both the right and the left handed users use the microwave easily.
